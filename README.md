# Pitiko Verbs

A mobile-first React app for practicing English irregular verbs.

## Current version

This first version includes:

- Flashcards for reviewing base, past and past participle forms
- A randomized quiz mode
- A searchable verb list
- A custom Pitiko Studios splash screen and color palette

## Running locally

Install dependencies:

```bash
npm i
```

Start the dev server:

```bash
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://127.0.0.1:5173
```

## Building

```bash
npm run build
```

The production build is created in `dist/`.
